# 🎮 Gaming Performance Analytics: End-to-End Free Fire Data Analysis

## 📌 Project Overview

This project analyzes Free Fire player performance data using SQL, Python, and Power BI. The objective is to identify player behavior patterns, win efficiency, map performance, KDA trends, and overall gameplay insights through interactive dashboards and data analysis.

## 🛠️ Tools & Technologies

* Power BI
* SQL Server
* Python (Pandas, NumPy, Matplotlib, Seaborn)
* CSV Dataset

## 📊 Dashboard Features

### Executive Overview

* Total Matches Played
* Average Rating
* Average KDA
* Average Damage
* Average Kills
* Match Distribution by Mode
* Matches Played by Map
* Match Outcome Breakdown
* Player Rating Score

### Player Performance Analysis

* Damage vs Rating Correlation
* KDA Performance by Game Mode
* Map Performance Rankings
* Performance Category Distribution
* Mode-wise Win Efficiency
* Dynamic Filters for Map, Mode, Result, and Performance Category

## 🗄️ SQL Analysis

* Match Performance Analysis
* Win Rate Calculation
* KDA Analysis
* Map-wise Performance Ranking
* Player Ranking Queries
* Aggregation and Grouping Analysis

## 🐍 Python Analysis

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Player Segmentation
* Performance Metrics Calculation
* Visualizations using Matplotlib and Seaborn

## 📈 Key Insights

* CS Ranked mode recorded the highest match participation.
* Bermuda was the most frequently played map.
* Special CS Ranked mode achieved the highest win efficiency.
* Higher KDA values generally resulted in better player ratings.
* Player performance categories helped identify elite and average performers.
